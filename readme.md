# Neo4j Tp

## Clone the repository
```bash
git clone https://github.com/Logards/neo4j-tp.git
cd neo4j-tp
```

## Install dependencies
```bash
pip install -r requirements.txt
```
## Create the db
```bash
docker compose up -d
```

## Run the project
```bash
python run.py
```

## test the project
```bash
python test.py
```
